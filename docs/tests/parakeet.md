## Оценка качества

- **Худшее качество на русском** среди всех моделей
- Текст частично бессмысленный, утрачены ключевые слова и контекст
- Модель оптимизирована для английского — русский поддерживается, но с заметно меньшей точностью
- Короткий выход: значительная часть 60 секунд не распознана вовсе
- Нет сегментации по времени

## Примечание

Parakeet TDT v3 заявляет поддержку 25 языков (включая русский), однако основной фокус модели — английский. На русском языке качество значительно ниже, чем у специализированных моделей (GigaAM, Whisper). Рекомендуется использовать для английского или в сочетании с другими моделями.

## Команда запуска

```bash
# Metal GPU (рекомендуется)
/usr/bin/time -l target/release/rustasr transcribe \
  --model models/parakeet-tdt-0.6b-v3 \
  --model-type parakeet \
  --audio tmp/hf_compare_31118765/full_16k_mono_60s.wav \
  --device metal \
  --out-text docs/tests/parakeet_transcription.txt

# CPU
/usr/bin/time -l target/release/rustasr transcribe \
  --model models/parakeet-tdt-0.6b-v3 \
  --model-type parakeet \
  --audio tmp/hf_compare_31118765/full_16k_mono_60s.wav \
  --device cpu \
  --out-text docs/tests/parakeet_transcription.txt
```
