## Оценка качества

- **Самый быстрый** — RTF 0.017 на Metal (2.8× быстрее CPU), абсолютный лидер среди всех моделей
- Хорошее качество на русском, правильные имена: «Алиной», «РТК», «МТС-партнёра»
- Есть пунктуация (запятые, вопросительные знаки, двоеточия)
- Заметные ошибки: «сц» вместо «стороне», лишние запятые
- Буква «ё» правильно используется (в отличие от Whisper)

```bash

  --model models/gigaam-v3-e2e-ctc \
  --model-type gigaam \
  --audio tmp/hf_compare_31118765/full_16k_mono_60s.wav \
```
